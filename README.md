# IMMT

IMMT 名字取自英文 "Input Method Mode Toggle" 首字母，主要功能是将 **Win + Space** 设置成切换 **微软中文输入法** 中/英状态的全局快捷键。

## 解决什么问题

Windows 10/11 自带的微软中文输入法，不能自定义切换中/英状态的快捷键，只提供了 **Ctrl + Space**、**Shift** 或者 **Ctrl** 这三个选项。

对于程序员群体，这三个快捷键选项都不是很合适。**Ctrl + Space** 被很多 IDE (VS, VS Code, Jetbrains) 触发补全的快捷键，**Shift** 容易在输入大写英文字符的时候容易被误触发，**Ctrl** 作为很多快捷键的前缀，也很容易误触发。

综上，这里选择用 **Win + Space** 作为切换 **微软中文输入法** 中/英状态的全局快捷键，解决默认快捷键冲突与误触发的问题。

## Q&A

Q: 支持哪些中文输入法？
A：暂时只支持系统自带的微软中文输入法。

Q：是否支持自定义快捷键绑定？
A：暂时还没支持，可以实现。
